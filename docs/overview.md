# 概要

GPURender は「会話動画」をターゲットにした GPU 描画寄りの最小レンダラ実装プロジェクトです。

このリポジトリは v0 では、下記の目的に限定します。

- `project.json` を正本として読み込む
- 2 人会話（左右配置キャラクター）と字幕帯を描画する
- 口パク・瞬き・背景演出を再現する
- 事前合成された音声と BGMを組み合わせた動画を `h264_nvenc` で書き出す

この段階では、既存の Remotion/Resolve パイプラインの代替ではなく、将来それらと接続しやすい独立 renderer として扱います。

## v0 の想定

- 固定レイアウト（`AgiDiscussion` 互換）
- 解像度 1920x1080、FPS 30 を想定
- レンダリングはフレーム画像列または `ffmpeg` 経由の動画化を前提

## 非対応（v0）

- エディタ UI
- Remotion JSX の完全互換
- Resolve タイムラインへの直接書き出し
- マルチテンプレートの汎用化

これらは次の開発フェーズで追加します。
